## How to Use (Simple 3-Step Process)

### Step 1: Install & Start (1 minute)
```bash
# Install dependencies
pip install fastapi uvicorn numpy scikit-learn matplotlib reportlab

# Authenticate Google Earth Engine
earthengine authenticate

# Start server
python -m backend.main_integrated_v4
```

### Step 2: Run a Scan (2 minutes)
```bash
curl -X POST http://localhost:8000/scan/complete \
  -H "Content-Type: application/json" \
  -d '{"latitude": 9.15, "longitude": -1.50, "commodity": "HC"}'
```

### Step 3: Review Results (5 minutes)
- Check JSON response for all 8 tier results
- ACIF confidence score (should be > 0.75 for TIER_1)
- Download PDF report with embedded visuals
- Review 2D/3D models and trap geometry
